Acoustic Music NYC

Acoustic Music NYC

I personally built a web app that contains all of NYC’s acoustic music events in one place. Each entry is searchable by content and tags. Users can submit events for moderators to approve and edit. Users can star their favorite events to create a personal calendar that they can view and share with friends. The app uses a Ruby on Rails backend paired with Vue.js frontend along with Google Maps JavaScript API to create the most user-friendly experience.

The website has been published online for beta texting while I rebuild it from scratch. I'm having friends submit actual content and give feedback.

Movie-Rater

Acoustic Music NYC

This is a movie rating app built as practice for a technical interview. It uses a Ruby on Rails backend that calls from a movie API hosted on RapidAPIs. Movies are added to the database by searching the API by title and clicking on a movie instance. You are then pushed to the movies show page where you can thumbs up or down a movie. The home page displays all the movies that have been added to the database ordered by thumbs ups.
